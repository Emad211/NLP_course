# Sentiment Labeling Guide (v1)

We label each comment about a doctor/clinic.

Labels:
-  1 = Positive (رضایت واضح)
-  0 = Neutral / Mixed / Informational (خنثی، ترکیبی، یا صرفاً اطلاعات)
- -1 = Negative (نارضایتی واضح)

Rules:
1) شکایت‌های سرویس/مطب هم Negative هستند:
   - معطلی، بی‌نظمی، منشی، پاسخگو نبودن، نوبت‌دهی، هزینه/پولکی، لغو نوبت و ...
2) Mixed:
   - اگر هم تعریف هم شکایت دارد:
     - اگر یکی واضحاً غالب است همان را بزن.
     - اگر واقعاً ترکیبی و مبهم است => 0
3) متن‌های صرفاً اطلاعاتی (بدون قضاوت):
   - مثل "ام آر آی نوشتن"، "عمل پلیپ" => 0
4) متن‌های خیلی کوتاه و مبهم:
   - مثل "خوب بود" => معمولاً 0 مگر کاملاً صریح باشد (مثل "افتضاح")
5) تمرکز روی تجربه و قضاوت متن است، نه تخصص یا موضوع درمان.
